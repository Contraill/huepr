      How Huepr works

      Huepr connects your Hyprland/Caelestia wallpaper changes to your browser. When you change your wallpaper, the native host detects it and pushes the new theme name to this extension. The extension then injects CSS custom properties into whitelisted sites — no page reload needed.

      Flow:

      wallpaper change → huepr-notify → theme_host.py → Native Messaging → background.js → content.js → :root CSS variables

      Themes

      Each theme is a named set of CSS variables. The theme name must exactly match the wallpaper filename stem (e.g., wallpaper Aurora.jpg → theme name Aurora).

      Built-in themes (always available): Aurora, Sunset, Latte, Frappe. Custom themes added via the Themes tab override built-ins.

      Standard theme variables:

      --theme-bg       main background
--theme-fg       primary text
--theme-surface  elevated surfaces (cards, sidebars)
--theme-accent   links, highlights
--theme-border   dividers
--theme-red      semantic red
--theme-green    semantic green
--theme-yellow   semantic yellow
--theme-blue     semantic blue

      If the current wallpaper has no matching theme, Huepr clears all injected variables and the site reverts to its default appearance.

      Sites & Mappings

      Add a site by its hostname (e.g. github.com). Then define mappings: each mapping connects a site CSS variable to a theme variable.

      Example: --color-canvas-default → --theme-bg means GitHub's canvas background will be set to the current theme's --theme-bg color.

      Finding a site's CSS variables

      Open DevTools on the target site and run this in the Console tab:

      const vars = new Set();
[...document.styleSheets].forEach(sheet => {
  try {
    [...sheet.cssRules].forEach(rule => {
      if (rule.style) [...rule.style].filter(p => p.startsWith('--')).forEach(p => vars.add(p));
    });
  } catch {}
});
console.log([...vars].sort().join('\n'));

      This prints all CSS custom properties defined in the site's stylesheets. Look for variables that control background, text, accent, and border colors.

      Alternatively: open DevTools → Inspector → select <html> → Computed tab → filter by --.

      Export / Import

      Export JSON (Sites tab) — downloads huepr-config.json with your whitelist and mappings.

      Export JSON (Themes tab) — downloads themes.json with your custom themes.

      Import JSON — imports the matching format depending on which tab is active.
